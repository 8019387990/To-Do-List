# To-Do-List

## Overview

The To-Do List App is a simple and responsive web application developed using **HTML**, **CSS**, and **JavaScript**. It helps users organize their daily tasks by allowing them to add, complete, and remove tasks through an intuitive and user-friendly interface.

This project demonstrates the fundamentals of front-end web development, including DOM manipulation, event handling, and responsive design.

---

## Features

- Add new tasks
- Mark tasks as completed
- Delete completed or unwanted tasks
- Responsive and clean user interface
- Fast and lightweight
- Beginner-friendly project

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Usage

1. Enter a task in the input field.
2. Click the **Add** button.
3. Click on a task to mark it as completed.
4. Delete tasks when they are no longer needed.
